# Note App

### Installation


```sh
$ npm install
$ node app.js --help
```
### Development

Want to contribute? Great!

Open your favorite Terminal and run these commands.

Add a note:
```sh
$ node app.js add --title="sourish" --body="sourish"
```
List all notes:
```sh
$ node app.js list
```
To read a note
```sh
$ node app.js read --title="sourish"
```
To delete a note
```sh
$ node app.js remove --title="sourish"
```
